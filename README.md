# Sentiment Analysis of Text Reviews

A complete end-to-end Natural Language Processing (NLP) project that classifies text reviews as **positive** or **negative**. The project starts with text cleaning and traditional Machine Learning models, then extends into Deep Learning with a Bidirectional LSTM, model evaluation, API serving, a simple web interface, and Docker-ready deployment files.

This repository is suitable for learning, portfolio demonstration, interview discussion, and as a foundation for a production-grade sentiment analysis service.

---

## Project Overview

Sentiment analysis is a foundational NLP task where the goal is to understand whether a piece of text expresses a positive or negative opinion. In this project, the text may come from movie reviews, product reviews, service feedback, or general customer comments.

The notebook walks through the full development lifecycle:

1. Load or create a review dataset.
2. Clean and normalize text data.
3. Explore the dataset.
4. Train classic ML models using TF-IDF features.
5. Compare Logistic Regression, Naive Bayes, and Linear SVC.
6. Tune the best classic ML model.
7. Build and evaluate a Bidirectional LSTM model.
8. Save trained models and preprocessing assets.
9. Generate a FastAPI backend for predictions.
10. Generate a simple web interface.
11. Generate Docker and deployment files.
12. Test final inference from saved model files.

## Dataset

The notebook supports two modes:

### Option 1: Use Your Own Dataset

Create a CSV file named:

```text
reviews.csv
```

The CSV should contain the following columns:

| Column | Description | Example |
|---|---|---|
| `review` | Text review | `This product is excellent and easy to use.` |
| `label` | Sentiment label | `positive` or `negative` |

Example:

```csv
review,label
This movie was fantastic and emotional,positive
The product broke after one day,negative
I loved the acting and story,positive
The service was very poor,negative
```

Place `reviews.csv` in the same folder as the notebook.

### Option 2: Use the Built-in Sample Dataset

If `reviews.csv` is not found, the notebook creates a small sample dataset automatically so that the full pipeline can run immediately.

For better accuracy, use a larger real-world dataset such as movie reviews, product reviews, or customer feedback.

## How to Run the Notebook

### Step 1: Clone or Download the Project

```bash
git clone <your-repository-url>
cd sentiment-analysis-text-reviews
```

Or download the files manually and open the folder in your IDE.

### Step 2: Create a Virtual Environment

For Windows PowerShell:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

For Windows Command Prompt:

```cmd
python -m venv .venv
.venv\Scripts\activate.bat
```

For macOS / Linux:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### Step 3: Install Jupyter

```bash
pip install notebook jupyterlab
```

### Step 4: Open the Notebook

```bash
jupyter notebook sentiment_analysis.ipynb
```

Or:

```bash
jupyter lab
```

### Step 5: Run All Cells

In Jupyter:

```text
Kernel → Restart & Run All
```

The notebook will install missing dependencies, train models, evaluate results, and generate deployment files.

## Common Problems and Fixes

### Problem: TensorFlow installation fails

Use Python 3.10 or 3.11 for best compatibility.

### Problem: Notebook runs slowly

Use the classic ML section first. LSTM training may take more time on CPU-only machines.

### Problem: API does not start

Make sure you are inside the generated project folder:

```bash
cd sentiment_analysis_project
```

Then run:

```bash
uvicorn app.main:app --reload
```
